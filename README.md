# sanaiyi-mobile

This is the mobile application skeleton for react-native and typescript

### Install Project

1. Clone project

```bash
git clone https://github.com/kbukum/rnts-skeleton.git
cd rnts-skeleton
```

2. Install Dependencies

```bash
npm install
```

### Join Development

1. Start npm :)

```bash
npm start
```

### Test Project

```bash
npm test
```
